# 🌟 AI角色聊天神器（语音版，代入感极强）

## 项目简介
这是一个**可以模拟任何角色的 AI 聊天软件（语音回复）**，支持各种主题和关系模式，让你沉浸在剧情之中，体验最真实、最上头的互动。  

无论是经典小说、影视剧，还是你自定义的角色与故事，都可以自由切换。你不仅可以单聊角色，还能 **组建多角色群聊**，同时享受 AI 生成的高品质语音，音色饱满、感情丰富，让每一次聊天都身临其境。

## 核心功能

### 1️⃣ 任意主题 & 角色
- 古典文学：红楼梦、三国、聊斋志异……  
- 现代影视：热门剧集、电影人物随意选  
- 自定义角色：你想的角色都能创建和聊天  

### 2️⃣ 多种关系模式
- 暧昧恋爱、友情、兄妹姐弟、父子母子  
- 三角关系、四角关系、复杂剧情演绎  
- 每一种关系都可以通过 AI 对话自然呈现  

### 3️⃣ 多角色群聊
- 同时与多名角色互动  
- 角色之间能互相回应、互动，剧情感极强  
- 模拟真实社交或剧情演绎，体验感爆棚  

### 4️⃣ AI语音聊天
- 高品质语音生成，音色自然动听  
- 角色说话带情绪，暧昧、温柔、调皮……全都有  
- 沉浸式体验，录屏展示也极具吸引力  

### 5️⃣ AI助手攻略
- 不会撩？不会演？AI助手帮你挑选话术、提供策略  
- 让每一次聊天都生动、上头、趣味十足  
- AI助手协助你管理主体和角色、自动创建群、聊天等
